Waltz

My Brackeys Jam 2020.2 Submission (My first one week gamejam) - First Person Interactive Film. Waltz is a detective who returns to a crime scene that no one has ever been able to solve. Unlike the others, he manages to use the power of Rewind to see things that were once there. Inside he discovers the whole truth, even the one he least expects.

Containment

"Containment" is a 3D first person shooter game for Windows. This game comes from a academic project designed and developed individually. Tests the player's speed and reflexes within a scary environment in which the user takes on the role of the hero.

Quinto Império

"Quinto Império" is a 2D top down role-playing game for Windows. An academic project made in order to sensitize the players about the civilized world day to day problems, hoping to change actions and mentalities.
